# Jellynix Plugin Repository

Public distribution host for the Jellynix Jellyfin plugin.

## Install

Add this manifest URL in Jellyfin under `Dashboard -> Plugins -> Repositories`:

`https://raw.githubusercontent.com/bttfw/jellynix-plugin-repository/main/manifest.json`

## Hosted Files

- `manifest.json`
- `packages/jellynix_0.1.0.4.zip`

The binary package is generated from the private source repository and published here for operator-facing plugin installation.
